# IMS Project

A school project for the IMS course focused on simulation/modeling in the field of ecology.

## Description
This repository contains an IMS project that combines multiple parts of the implementation and documentation.  
The project includes source code, Python scripts, documentation files, a Makefile, and generated outputs.

Main repository contents:
- `src/` – source files
- `py/` – Python scripts
- `doc/` – project documentation
- `Makefile` – build automation
- output and archive files

## Build
Compile the project using:

make

## Run
After building, run the program according to the generated executable or the project instructions in the documentation.

## Notes
The repository also contains a PDF documentation file and example outputs related to the simulation.

## Authors
Project created as part of the IMS course.
